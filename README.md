# ATLAS Motors

ATLAS Motors is a front-end web development project designed to showcase a collection of high-end exotic vehicles through a luxury dealership experience. Inspired by premium automotive brands, the website emphasizes elegant design, immersive media, and user interaction.

## Features

* Luxury-inspired homepage experience
* Hypercar and supercar collections
* Dedicated vehicle detail pages
* Interactive image galleries
* Vehicle inquiry modal
* Responsive design
* Inventory sorting functionality
* Custom branding and visual identity

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Goal

The goal of this project is to demonstrate front-end development skills through the creation of a polished, professional automotive website that balances aesthetics, functionality, and user experience.
